# Skija 入门指南

Skija 将 Skia 图形引擎的强大功能带到了 Java 虚拟机。本指南提供了使用 Skija 开始绘图的高级概述。

## 核心概念

在深入代码之前，了解 Skija 生态系统中的主要角色会很有帮助：

- **Surface（表面）**：绘制的目标（就像一张纸）。
- **Canvas（画布）**：用于执行绘图操作的接口（就像你的手）。
- **Paint（画笔）**：定义绘制内容的颜色、样式和效果（就像你的钢笔或画笔）。

## 快速入门路径

要在屏幕上或图像文件中显示你的第一个“Hello World”，我们建议按照以下步骤操作：

1.  **[安装](installation.md)**：为你的特定平台（Windows、macOS 或 Linux）设置项目依赖项。
2.  **[渲染基础](rendering-basics.md)**：学习如何创建一个简单的内存 Surface 并绘制你的第一个基本形状。
3.  **[排版](typography.md)**：使用系统字体或自定义字体文件为你的创作添加文本。
4.  **[GPU 渲染](gpu-rendering.md)**：对于交互式应用程序，学习如何利用显卡的强大功能。

## 资源管理

Skija 是一个围绕 C++ 库的高性能包装器。虽然它会自动为你管理内存，但理解[资源管理](resource-management.md)原则对于构建健壮的应用程序是推荐的。
